# Tienda WhatsApp Pro - Configuración
# =====================================
# Este archivo contiene la configuración del proyecto.
# Copia las variables necesarias a tu archivo .env.local

## Variables de Entorno Requeridas:

### Supabase
# NEXT_PUBLIC_SUPABASE_URL=https://tu-proyecto.supabase.co
# NEXT_PUBLIC_SUPABASE_ANON_KEY=tu_anon_key_aqui

### Google Maps (para autocompletado de direcciones)
# NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=tu_api_key_google_maps

### WhatsApp - Números de teléfono (sin el +)
# NEXT_PUBLIC_WHATSAPP_TIENDA=982432561
# NEXT_PUBLIC_WHATSAPP_ADMIN=982432561

## Nota sobre WhatsApp:
# - WHATSAPP_TIENDA: Número donde los clientes envían sus pedidos
# - WHATSAPP_ADMIN: Número del admin para recibir notificaciones de nuevos pedidos
#   (puede ser el mismo número o uno diferente)
